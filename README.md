# Terminal-Bang-like-Game
Мультиплеерная карточная игра, доступная через SSH. Пользователь авторизуется по SSH-ключу, попадает в TUI-интерфейс, создаёт/выбирает комнату, играет матч с другими игроками, результаты сохраняются в профиль. Проект реализует собственный SSH gateway, терминальный UI, матчмейкинг, игровую машину, reconnect-механику, статистику и удобный деплой
